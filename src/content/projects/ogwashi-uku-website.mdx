# Ogwashi-Uku Website

A modern website and member portal for the Ogwashi-Uku USA Association: public-facing pages plus a secured backend for membership, chapters, events, and admin workflows.

## What it includes
- Public site: chapters, events, impact, donate, contact
- Portal: authentication + role-based access (national admin, chapter admin, member)
- Data-backed APIs and a real database (not demo content)
